# koa-static-scaffold
a scaffold for develop and build static website fast.

## Install
```
$ npm i -g koa-static-scaffold
```

## Bootstrap
```
$ scaffold demo
```

## Usage

### Preview && develop

change to your 'demo' dir

```sh
$ cd demo
```

install node deps
```
$ npm i
```

start local server
```
$ node --harmony app.js
```

visit `http://localhost:8300`
